This is an AI-powered image coloring web application built using **Python** and **Streamlit**.

### Features
- Upload black & white images
- Automatic AI-based coloring
- Download the colored image
- Simple and clean web interface

### Tech Stack
- Python
- Streamlit
- PIL / OpenCV
- Deep Learning
